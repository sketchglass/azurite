# azurite

Casual Painting App in Electron

## Prerequisites

* Node.js
* yarn
* typings

### Windows

* Python 2.7 / Visual C++ (for building native modules with node-gyp)

## Build

```
yarn install
typings install
npm run watch
```

## Run App

```
npm run app
```

## Package app

```
npm run package
```

The packaged app will be in `/build`.
